# Pocket CTO archive

This directory holds historical Pocket CTO material.

It is valuable because the old repo contains strong work on mission orchestration, replay, evidence, approvals, runtime integration, and the first twin/query surfaces.

It is **not** the active product scope anymore.

## What belongs here

- historical `plans/EP-*.md` files
- M2 and M3 exit reports
- roadmap snapshots
- engineering-era milestone notes
- selected engineering architecture notes once they have active Pocket CFO replacements

## How to use the archive

Read it for:

- reusable control-plane patterns
- replay and evidence ideas
- connector implementation references
- lessons from milestone execution

Do not read it as the current answer to:

- product boundary
- source of truth
- mission taxonomy
- artifact vocabulary
- operator UX goals

Those answers now live in the active Pocket CFO docs.
