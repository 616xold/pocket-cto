Rationale: no remote MCP before host/network/security review and a later Finance Plan.

Decision: Apps SDK iframe/resource implementation is deferred.
Rationale: no Apps SDK iframe/resource before a UI/resource security plan and a later implementation Finance Plan.

Decision: public app implementation, app submission/listing/public assets, listing copy, and app-submission artifacts are deferred.

Decision: prompt injection is treated as untrusted data.
